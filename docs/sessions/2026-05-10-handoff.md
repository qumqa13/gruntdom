## 7. Key decisions (ADR-0002 highlights)

| Decision | Owner | Rationale |
|---|---|---|
| Plan B variant (a): Cesium open-source + Geoportal NMT terrain + ortofoto + ULDK polygon, NO OSM extruded buildings | ADR-0002 §2.1 | Atelier identity > photorealism; OSM blocks clash visually with clay polygon |
| 3-tier terrain ladder: Polish NMT (Tier 1) → Cesium ION (Tier 2) → Ellipsoid (Tier 3 = ADR-0003) | ADR-0002 §6.3 | Bounded fallback, no surprise degradation |
| Showcase-first: F2 = Balice 773 only; others fallback-2d until F2.5 | ADR-0002 §2.2 | One polished demo > four mediocre |
| Three-step cross-validation gate: ≤2% silent / 2-5% soft seller-accept / >5% admin review | ADR-0002 §2.5 | Match friction to risk |
| Headless Puppeteer screenshots cached in R2, frozen in time | ADR-0002 §2.4 | Stronger trust signal than "click here to check now" |
| Sibling-map provenance (`provenance?: ProvenanceMap<T>`) over `Provenanced<T>` wrapper | F1-T1 implementation | Migration-additive, zero existing-consumer changes |
| Cost cap $30/30d TOTAL all services | ADR-0002 §2.6 | Replicate ~$5-10 dominant; everything else free tier |
| Geoportal compliance: User-Agent identification, rate limit 1/30s/layer/IP, cache TTL ≥ 7d, pre-launch GUGiK email | ADR-0002 §2.8 | ToS prohibits mass automated harvesting |
| License: PZGiK = "Otwarte dane GUGiK" per Ustawa Prawo Geodezyjne 2020 (Dz.U. 2020 poz. 782), NOT formal CC BY 4.0 | LICENSES.md | Precise law citation matters; Oskar initially assumed CC BY 4.0 |
