# Gruntdom — Redesign landing page

**Data:** 2026-05-04
**Scope:** Strona główna `/` + drobne zmiany w `Header` i `Footer`. Detail page `/plots/[slug]` poza zakresem.

## Cel

Hero strony głównej dostaje cinematic video tła (proces transformacji pustej działki w wizualizację zabudowy). Reszta sekcji zostaje przeprojektowana wokół jednej narracji: **"Zobacz, co naprawdę można zbudować na działce, zanim ją kupisz."** — produkt jako narzędzie analizy potencjału, nie jako kolejny portal ogłoszeniowy.

## Kierunek wizualny

**Architectural Intelligence** — kontynuacja istniejącej palety Atelier (paper #F4F1EA, ink, clay, moss), z mocniejszą skalą hero i bardziej cinematic motion. Nie odchodzimy od Fraunces / Inter / JetBrains Mono — to jest distinctive.

**Hero treatment:** video w paper-frame container (nie fullscreen). Typografia i przestrzeń dookoła video są częścią kompozycji — jak okładka magazynu architektonicznego, gdzie obraz jest ramowany przez świadomy biały margines, nie wypełnia całego ekranu.

## Język motion

| Element | Animacja |
|---------|----------|
| Hero text | Existing fade-in + y, atelier ease, staggered |
| Hero stats | Animated counter 0 → target (Framer `useTransform`) |
| Video | autoplay, muted, loop, playsinline, `object-cover` |
| Section reveals | `Reveal` / `RevealStagger` (existing) |
| Process timeline | `useScroll` + `useTransform` na progress line, dots aktywują się przy scroll-progress |
| Showcase tabs | `AnimatePresence` z `mode="wait"`, `motion.div` crossfade |
| UseCases cards | Native CSS scroll-snap, hover translateY -4px |
| CTA arrows | translateX +1 na hover (existing) |

## Czego nie robimy (anti-patterns)

- ❌ Fullscreen video z dark overlay → tracimy paper warmth, generic SaaS
- ❌ Glassmorphism / backdrop-blur na kartach
- ❌ Gradient text na headlinach
- ❌ Particle systems / sparkle effects
- ❌ Scroll-jacking / pinned hero
- ❌ Floating chatbot / pulsujące CTA buttony
- ❌ Lucide icons w komponentach nawigacyjnych (custom SVG line-art)
